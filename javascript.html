JAVASCRIPT - INTRODUÇÃO

É uma das 3 linguagens de programação fundamental para o desenvolvimento de websites. HTML para definir o conteúdo, CSS para especificar o layout e JAVASCRIPT para interagir com o conteúdo.

O código JS é inserido entre as tags <script> e </script> no HTML. Exemplos antigos de JS podem usar um atributo type:
<script type="text/javascript">
O atributo type não é obrigatório. JS é a linguagem padrão em HTML.

Pode ser inserido: SCRIPT INTERNO dentro de BODY OU HEAD ou SCRIPT EXTERNO utilizando arquivos .JS. Um script externo pode ser referenciado de três maneiras:
1. URL completa (scr="http://exemplo.com/js/script.js");
2. Caminho de arquivo: scr="js/script.js";
3. Sem nenhum caminho: scr="script.js".
NOTA: SCRIPTS EXTERNOS NÃO PODEM CONTER <script>!
Vantagens de script externo:
1. Separa o HTML do código;
2. Torna o HTML e JS mais fáceis de ler e manter;
3. Aumenta a velocidade de carregamento.
Para adicionar vários arquivos de script, use várias tags de script.

O QUE PODE FAZER?

1. Pode alterar o conteúdo HTML;
2. Pode alterar atributos HTML;
3. Pode alterar estilos CSS;
4. Pode ocultar/exibir elementos HTML.

Um function() é um bloco de código JS, que pode ser executado quando "chamado". Por exemplo, uma função é chamada quando ocorre um evento (como um clique).

O resultado dessa função pode ser exibido de diferentes maneiras:
1. Escrevendo em um elemento HTML innerHTML;
2. Escrevendo na saída HTML document.write();
3. Escrevendo em uma caixa de alerta alert();
4. Escrevendo no console console.log().

INNERHTML

Para acessar um elemento HTML, o JS pode usar o método: document.getElementById(id)
O atributo id define o elemento HTML. A innerHTML define o conteúdo HTML. Exemplo: 

document.getElementById("demo").
innerHTML = 5 + 6;

DOCUMENT.WRITE()

Para fins de teste, é conveniente usar.
USAR APÓS O CARREGAMENTO DE UM DOCUMENTO, EXCLUIRÁ TODO O HTML EXISTENTE! USAR APENAS PARA FINS DE TESTE! 

document.write(5 + 6);

CONSOLE.LOG()

Para fins de depuração, pode usar o método console.log para exibir dados. 

console.log(5 + 6);

IMPRESSÃO JS

JavaScript não possui nenhum objeto ou método de impressão. Você não pode acessar dispositivos de saída em JS. A única exceção é que você pode chamar o window.print() para imprimir a janela atual. 

<button onlick="window.print();">
IMPRIMIR
</button>

DECLARAÇÕES JAVASCRIPT

Um programa de computador é uma lista de "instruções" a serem executadas. Um código JS é uma lista de instruções. Em HTML, os programas JS são executados pelo navegador. As instruções são compostas por:
1. VALORES;
2. OPERADORES;
3. EXPRESSÕES;
4. PALAVRAS-CHAVES;
5. COMENTARIOS.

PONTO E VÍRGULA separam instruções JS. Quando separadas por ponto e vírgula, são permitidas múltiplas instruções em uma linha. Não é obrigatório, mas É ALTAMENTE RECOMENDADO.

ESPAÇO EM BRANCO JS ignora espaços em branco. Você pode adicionar para torná-lo legível. Uma boa prática é: colocar espaços ao redor dos operadores (= + - *)

BLOCOS DE CÓDIGO instruções podem ser agrupadas em blocos de códigos, entre chaves { }. O objetivo é definir instruções a serem executadas juntas. Exemplo: 

function myFunction() {
  document.getElementById("demo1").
  innerHTML = "Olá!";
  document.getElementById("demo2").
  innerHTML = "Oi.";
}

PALAVRAS CHAVES as instruções geralmente começam com uma palavra-chave para identificar a ação a ser executada. Exemplo:

var /// declara uma variável;
let /// declara uma variável de bloco;
const /// declara uma constante de bloco;
if /// declara um bloco de condição;
switch /// declara um bloco de instruções a serem executadas em diferentes casos;
for /// declara um bloco de instruções para repetições;
function /// declara uma função;
return /// declara um retorno de função;
try /// declara um bloco de instruções para tentar executar.

SINTAXE

Conjunto de regras para construir o JS.

// CRIAR VARIÁVEIS:
var x;
let y;

// USAR VARIÁVEIS:
x = 5;
y = 6;
let z = x + y;

VALORES a sintaxe JS define dois tipos de valores: variáveis e fixos. Valores fixos são chamados de LITERAIS e valores das variáveis são chamados de VARIÁVEIS.
>>>> LITERAIS:
As duas regras mais importantes para valores fixos são:
1. Números são escritos com ou sem decimais: 10.50 // 1001.
2. Strings são textos escritos entre aspas duplas ou simples: "John Doe" // 'John Doe'.
>>>> VARIÁVEIS:
São usadas para armazenar valores de dados. JS usa as palavras: var, let e const para declarar variáveis.

OPERADORES ARITIMÉTICOS ( + - * / ) são usados para calcular valores. E para atribuir valores: é usado o sinal de igual.

IDENTIFICADORES são nomes JS, usados para nomear variáveis, palavras-chaves, funções, etc. As regras são as mesmas na maioria das linguagens:
Deve começar com uma letra (AZ ou az), cifrão ($) ou sublinhado (_)
