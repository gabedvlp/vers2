TIPOS DE OPERADORES

Existem diferentes tipos de operadores:

  • Aritméticos
    • OPERADOR DESCRIÇÃO
    + Adição
    - Subtração
    * Multiplicação
    ** Exponenciação
    / Divisão
    % Módulo (resto da divisão)
    ++ Incremento
    -- Decremento
  • De atribuição
    • OPERADOR EXEMPLO IGUAL A
    = x = y x = y
    += x += y x = x + y
    -= x -= y x = x - y
    *= x *= y x = x * y
    /= x /= y x = x / y
    %= x %= y x = x % y
    **= x **= y x = x ** y
  • De comparação
    • OPERADOR DESCRIÇÃO
    == Igual a
    === Valor/tipo iguak
    != Não igual
    !== Valor/tipo não igual
    > Maior que
    < Menor que
    >= Maior ou igual que
    <= Menor ou igual que
    ? Ternário
  • De string
    • Todos os operadores de comparação acima podem ser usados em strings.
    • Adição = Concatenar
  • Lógicos
    • OPERADOR DESCRIÇÃO
    && Lógico e
    || Lógico ou
    ! Lógico não
  • Bit a bit
    • OPERADOR DESCRIÇÃO
    & AND
    | OR
    ~ NOT
    ^ XOR
    << Deslocamento para esquerda
    >> Deslocamento para direita
    >>> Deslocamento para direta sem sinal
  • Ternários
  • São uma forma condicional de expressão para avaliar uma expressão e retornar um valor com base em uma condição verdadeira ou falsa. Um if-else de uma linha.
  • De tipo
    • OPERADOR DESCRIÇÃO
    typeof Retorna o tipo de uma variável
    instanceof Retorna true se um objeto for uma instância de um tipo de objeto.
