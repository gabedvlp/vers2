TABELAS <TABLE>

As tabelas HTML permitem que os desenvolvedores da web organizem os dados em linhas e colunas.

Cada célula da tabela é definida por um par de tags <td> (dados da tabela - colunas). Tudo entre elas é o conteúdo da tabela.

Cada linha da tabela é definida por um par de tags <tr> (linha da tabela). Você pode ter quantas linhas quiser em uma tabela; apenas certifique-se de que o número de células seja o mesmo em cada linha.

Para adicionar um cabeçalho, use a tag <th>.

Tag Description
<table> Define uma tabela
<th> Define uma célula de cabeçalho em uma tabela
<tr> Define uma linha em uma tabela
<td> Define uma célula em uma tabela
<caption> Define uma legenda para a tabela
<colgroup> Especifica um grupo de uma ou mais colunas em uma tabela para formatação
<col> Especifica propriedades de coluna para cada coluna dentro de um elemento <colgroup>
<thead> Agrupa o conteúdo do cabeçalho em uma tabela
<tbody> Agrupa o conteúdo do corpo em uma tabela
<tfoot> Agrupa o conteúdo do rodapé em uma tabela

LISTAS <UL> e <OL>

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Lista NÃO Ordenada começa com a <ul>. Cada item da lista começa com a <li>.

  • Item 1
  • Item 2
  • Item 3

Lista Ordenada começa com a <ol>. Cada item da lista começa com a <li>.

  1. Item 1
  2. Item 2
  3. Item 3

Lista de Descricão é uma lista de termos, com uma descrição de cada um. A tag <dl> é usada para definir uma lista de descricão, a tag <dt> é usada para definir o termo e a tag <dd> é usada para definir a descrição.

Termo 1
Descricão do termo 1
Termo 2
Descricão do termo 2
