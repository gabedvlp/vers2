CSS - INTRODUÇÃO

É a linguagem que usamos para estilizar uma página da Web.

    1. Significa "FOLHAS DE ESTILO EM CASCATA";
    2. Descreve como os elementos HTML devem ser exibidos;
    3. Pode controlar o layout de várias páginas de uma vez;
    4. CSS externos são armazenados em arquivos .CSS

O HTML NUNCA foi planejado para conter tags para formatar uma página da web! Quando tags como <font> e atributos de cor foram adicionados à especificação HTML 3.2, isso iniciou um pesadelo para os desenvolvedores web. O desenvolvimento de grandes sites, onde fontes e informações de cores eram adicionadas a cada página, tornou-se um processo longo e caro.

Para resolver este problema, o World Wide Web Consortium (W3C) criou o CSS e removeu a formatação de estilo da página HTML.

SINTAXE CSS

Uma regra CSS consiste em um seletor e um bloco de declaração. Exemplo de um CSS: 

    p {
      color: red;
      text-align: center;
    }

1. p: é um seletor em CSS (aponta para o elemento HTML que você deseja estilizar: <p>);
2. color: é uma propriedade e red é o valor da propriedade;
3. text-align: é uma propriedade e center é o valor da propriedade.

SELETORES

Os seletores CSS são usados ​​para “encontrar” (ou selecionar) os elementos HTML que você deseja estilizar. Podemos dividir os seletores CSS em cinco categorias:
1. SELETORES SIMPLES: Selecionam elementos com base no nome, id ou classe.
2. SELETORES COMBINADORES: Selecionam elementos com base em um relacionamento específico entre eles.
3. SELETORES DE PSEUDOCLASSE: Selecionam elementos com base em um determinado estado.
4. SELETORES DE PSEUDOELEMENTOS: Selecionam uma parte de um elemento.
5. SELETORES DE ATRIBUTOS: Selecionam elementos com base em um ATRIBUTO ou VALOR.

SELETOR DE ID

Usa o atributo id para selecionar um elemento específico. O id de um elemento é ÚNICO em uma página, e o id deve ser exclusivo para cada elemento. Para selecionar um elemento com id, escreva um caractere hash (#), seguido do id do elemento. Exemplo: 

    #header {
      color: red;
      text-align: center;
    }

NOTA: UM NOME DE IDENTIFICAÇÃO NÃO PODE COMEÇAR COM UM NÚMERO.

SELETOR DE CLASSE

Usa o atributo class para selecionar um elemento com base em uma lista de classes. Para selecionar um elemento com classe, escreva o caractere de ponto (.) seguido do nome da classe. Exemplo:

    .center {
      text-align: center;
      color: red;
    }

Você também pode especificar que apenas elementos HTML específicos devem ser afetados por uma classe. Exemplo:

  p.center {
      text-align: center;
      color: red;
    }

Neste exemplo, apenas os elementos <p> com class="center" serão afetados.

Os elementos HTML também podem referir-se a mais de uma classe. Exemplo:

  <p class="center large">DUAS CLASSES.</p>

Neste exemplo, os elementos <p> com class="center" e class="large" serão afetados.

NOTA: UM NOME DE CLASSE NÃO PODE COMEÇAR COM UM NÚMERO.

SELETOR UNIVERSAL

O seletor universal (*) seleciona todos os elementos HTML. Exemplo: 

  * {
      color: red;
      text-align: center;
    }
A regra afetará todos os elementos HTML da página.
