HIPERLINKS

Links HTML são hiperlinks. Você pode clicar em um link e ir para outro documento. Não precisa ser especificamente um texto, pode ser uma imagem ou qualquer outro elemento.

SINTAXE

A tag <a> define um hiperlink. A sintaxe é:
<a href="url">texto</a>
O atributo mais importante é o href, que define o destino do link.

ATRIBUTO DE DESTINO

Por padrão, a página vinculada será exibida na janela atual. Para alterar isso, você deve especificar outro destino para o link:
<target> especifica onde abrir o documento e pode ter um dos seguintes valores:
1. _self: padrão;
2. _blank: abre em uma nova janela ou guia;
3. _parent: abre o documento no quadro pai;
4. _top: abre o documento em todo o corpo da janela.

BOTÕES, IMAGENS E E-MAIL

Você pode envolver os elementos dentro da tag <a> ou adicionar um código JS. (onclick="-").

Use "mailto:email@email.com" dentro de href para criar um link que abra o e-mail do usuário.
