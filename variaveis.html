DECLARANDO VARIÁVEIS

São contêiners para armazenamento de dados. Podem ser declaradas de quatro maneiras:

  1. Usando a palavra-chave var;
  2. Usando a palavra-chave let;
  3. Usando a palavra-chave const;
  4. Automáticamente.
NÃO DECLARADO:
<script>
x = 5;
y = 6;
z = x + y;
<script> 
Eles são declarados a primeira vez quando usados.
RESULTADO:

OBSERVAÇÃO: Sempre declarar as variáveis antes.

a palavra-chave var é o mais antigo e só deve ser usado em código escrito para navegadores mais antigos. Se essa não é a intenção, prefira usar as palavras-chaves let ou const.

USANDO VAR:
<script>
var x = 5;
var y = 6;
var z = x + y;
</script>
 USANDO LET:
<script>
let x = 5;
let y = 6;
let z = x + y;
</script>
USANDO CONST:
<script>
const x = 5;
const y = 6;
const z = x + y;
</script>
 EXEMPLO MISTO:
<script>
const p1 = 5;
const p2 = 6;
let ttl = p1 + p2;
</script>

As variáveis p1 e p2 foram declaradas com const. Esses valores são CONSTANTES, não podem ser alterados. A variável ttl foi declarada com let. Esses valores podem ser alterados.

Quando usar VAR, LET ou CONST?

  1. Use const se o VALOR e o TIPO (Arrays e Objetos) não deve ser alterado;
  2. Use let se não puder usar const;
  3. Somente para suporte a navegadores antigos.

IDENTIFICADORES

Todas as variáveis precisam ser identificadas com um nome exclusivo. Podem ser nomes curtos (x, y) ou mais descritivos (idade, soma). Regras gerais:

  1. Letras, dígitos, sublinhados e cifrões;
  2. Devem começar com letra;
  3. Podem começar começar com cifrão e sublinhado;
  4. Diferenciam maiúsculas e minúsculas;
  5. Palavras reservadas (como palavras-chaves) não podem ser usadas.

ATRIBUINDO VALORES

O sinal de igual (=) atribui um valor a uma variável, diferente da álgebra, não é "igual a".
OBSERVAÇÃO: "igual a" é escrito como == .
let carName;
carName = volvo
ou
let carName = volvo

Você pode declarar muitas variáveis em uma instrução separando-as com vírgula. Ou declarar sem valor. O valor pode ser algo que precisa ser calculado ou algo que será fornecido posteriormente, como a entrada de um usuário.

Não é possível redeclarar uma variável com let ou const, mas se você redeclarar uma variável declarada com var, ela não perderá seu valor.

DIFERENÇAS LET, CONST E VAR

var let const
Global Escopo de bloco Escopo de bloco
Redeclara Em outro bloco Em outro bloco
Reatribui Reatribui Não reatribui
Içado Não é içado Não é içado
Vincula a this Não vincula a this Não vincula a this
  1. ESCOPO DE BLOCO: Não pode ser acessada de fora do { } que foi declarada.
  2. GLOBAL: Acessada no código inteiro.
  3. IÇADA: Movida para o topo, antes da execução real.
  4. THIS: Palavra-chave definida dentro do contexto de execução de uma função.

OBJETOS E MATRIZES CONST

A palavra-chave const é um pouco enganosa, ela não define um valor constante, define uma REFERÊNCIA CONSTANTE a um valor. Por causa disso não pode reatribuir um valor, uma matriz ou um objeto constante, mas pode alterar os elementos da matriz e propriedades do objeto constante.

// Array constante:
const cars = ["Saab", "Volvo", "BMW"];
// Pode mudar um elemento:
 cars[0] = "Toyota";
// Pode adicionar um elemento:
 cars.push("Audi");
// Objeto constante:
 const car = {type:"Fiat", model:"500", color:"white"};
// Pode mudar a propriedade:
 car.color = "red";
// Pode adicionar uma propriedade:
 car.owner = "Johnson";
