HTML - INTRODUÇÃO

<!DOCTYPE html>
<html>
  
<head> 
  <title>TÍTULO</title> 
</head>
  
<body> 
  <h1>CABEÇALHO</h1>
  <p>PARÁGRAFO</p> 
</body>
</html>

Um elemento HTML é definido por uma TAG INICIAL, algum CONTEÚDO e uma TAG FINAL. O elemento HTML é tudo, desde a tag inicial até a tag final.

NOTA: ALGUNS ELEMENTOS NÃO POSSUEM CONTEÚDO, COMO O ELEMENTO <br>. SÃO CHAMADOS ELEMENTOS VAZIOS E NÃO POSSUEM TAG FINAL.

LINKS HTML:

São definidos com a tag <a>: 

<a href="https://www.w3schools.com">link</a>
1. Destino do link: HREF (usado para links e direcionamento).

IMAGENS HTML:

São definidas com a tag <img>: 

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

1. Arquivo de origem: SRC (usado para recursos incorporados)
2. Texto alternativo: ALT (usado para acessibilidade)
3. Tamanho: WIDTH (largura) e HEIGHT (altura)

VISUALIZAR FONTE HTML:

VER CÓDIGO-FONTE HTML:
CTRL + U em uma página

INSPENCIONAR UM ELEMENTO HTML:
F12 em uma página ou Botão direito sobre um elemento específico - inspecionar

ATRIBUTOS HTML:

Todos os ELEMENTOS podem ter atributos, eles fornecem informações adicionais sobre os elementos e são SEMPRE especificados na TAG INICIAL.

Geralmente vêm em pares nome/valor como: nome="valor" e NÃO diferencia MAÍUSCULAS de MINÚSCULAS. A recomendação é usar SEMPRE letras minúsculas em HTML. Atributos precisam ser citados em aspas simples ou duplas:
Aspas duplas são mais comuns, aspas simples são utilizadas quando o próprio valor do atributo contém aspas duplas. Exemplo:
<p title='John "ShotGun" Nelson'> ou
<p title="John 'ShotGun' Nelson">

TÍTULOS, PARÁGRAFOS E TEXTO PRÉ-FORMATADO:

1. Títulos são definidos com as tags "h1" a "h6":
> H1: Título mais importante;
> H6: Título menos importante.
1.1. Os navegadores adicionam automáticamente uma margem antes e depois de um título;
1.2. Mecanismos de pesquisa usam os títulos para indexar a estrutura e o conteúdo;
1.3. Usuários folheam uma página pelos títulos;
1.4. USE APENAS PARA TÍTULOS.

2. Parágrafos são definidos com a tag "p":
> Sempre começa uma nova linha e geralmente é um bloco de texto.
2.1. Os navegadores adicionam automáticamente uma margem antes e depois;
2.2. Não se tem certeza de como será exibido:
> Telas grandes ou pequenas e janelas redimensionadas = resultados diferentes.
2.3. Não é possível alterar a exibição com espaços ou linhas:
> O navegador removerá automáticamente qualquer espaço ou linha adicional;
> A tag "hr" é usada para separar o conteúdo em uma página HTML. Define uma quebra temática e geralmente é exibida como uma regra horizontal.
> A tag "br" define uma quebra de linha sem iniciar um novo parágrafo.
> NOTA: AS TAGS HR E BR SÃO TAGS VAZIAS 

3. Textos pré-formatados são definidos com a tag "pre":
> Preserva espaços e quebras de linha;
> É exibido em uma fonte de largura fixa (geralmente Courier).

FORMATAÇÃO E CITAÇÃO:

1. Formatação de texto:
<b> Define texto em negrito, sem qualquer importância extra.
<strong> Define texto de grande importância. Normalmente é exibido em negrito.
<i> Costuma ser usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, o nome de um navio etc. Normalmente é exibida em itálico.
<em> Texto enfatizado. Normalmente exibido em itálico. Um leitor de tela pronunciará as palavras usando ênfase verbal.
<mark> Texto marcado ou destacado.
<small> Texto menor.
<del> Texto excluído de um documento. Os navegadores geralmente traçam uma linha.
<ins> Texto inserido em um documento. Os navegadores geralmente sublinharão.
<sub> Texto subscrito. Aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor.
<sup> Texto sobrescrito. Aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor.

2. Citações:
<blockquote> Define uma seção citada de outra fonte. Navegadores geralmente recuam.
<q> Citações curtas. Normalmente inserem aspas ao redor.
<abbr> Abreviações ou acronimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Podem fornecer informações úteis aos navegadores, sistemas de tradução e motores de busca. 

NOTA: USE "TITLE" PARA MOSTRAR A DESCRIÇÃO AO PASSAR O MOUSE.

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

<address> Informações de contato do autor/proprietário de um documento ou artigo. Podem ser: e-mail, site, endereço físico, telefone, rede social etc. Normalmente aparece em itálico, sempre adiciona uma quebra de linha antes e depois.
<cite> Título de um trabalho criativo. Livro, poema, filme, pintura, filme etc. O nome de uma pessoa não é título de uma obra. Geralmente aparece em itálico.
<bdo> Substituição bidirecional. Usada para substituir a direção atual do texto. Normalmente exibido em itálico.
