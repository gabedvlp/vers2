Cada elemento HTML possui um valor de exibição padrão, dependendo do tipo de elemento. Os dois valores de exibição mais comuns são bloco e embutido.

BLOCOS

Um elemento em nível de bloco sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço (uma margem) antes e depois do elemento. sempre ocupa toda a largura disponível (estende-se para a esquerda e para a direita tanto quanto possível).

Dois elementos de bloco comumente usados ​​são: <div> define uma divisão/seção;
<p> define parágrafo.

<address> <article> <aside> <blockquote> <canvas> <dd> <div> <dl> <dt> <fieldset> <figcaption> <figure> <footer> <form> <h1> - <h6> <header> <hr> <li> <main> <nav> <noscript>  <ol> <p> <pre> <section> <table>  <tfoot> <ul> <video>

EMBUTIDOS

Um elemento embutido não começa em uma nova linha. Ocupa apenas a largura necessária.
Nota: Um elemento inline não pode conter um elemento de nível de bloco! 


<a> <abbr> <acronym> <b> <bdo> <big> <br> <button> <cite> <code> <dfn> <em> <i> <img> <input> <kbd> <label> <map> <object> <output> <q> <samp> <script> <select> <small> <span> <strong> <sub> <sup> <textarea> <time> <tt> <var>

DIV e SPAN:
DIV é usado como contêiner para outros elementos. SPAN é usado para partes do texto ou documento.

<div> como contêiner

O elemento é frequentemente usado para agrupar seções de uma página da web.
ALINHAR CENTRALMENTE: Se você tiver um que não tenha 100% de largura, defina margin: auto
ALINHAR LADO A LADO: Existem diferentes métodos, os mais comuns são:
1. float
2. display: inline-block
3. display: flex

Para fazer o método CSS flex funcionar, envolva os elementos com outro <div> e dê a ele o status de flex container. Exemplo: 

.conteiner {
display: flex;
}
.conteiner > div {
width: 20%;
}

4. display: grid

Parece quase igual a flex, mas tem a capacidade de definir mais de uma linha e posicionar cada linha individualmente. Dê a ele o status de grid e defina a largura de cada coluna. Exemplo: 

.conteiner {
display: grid;
grid-template-columns: 20% 20% 20% 20% 20%
}
